# Project Documentation

This `docs/` folder contains reference material and implementation notes for the SMS-Aksh / Accord Living project.

## Files

- `HACKATHON_DEMO_ROADMAP.md` – High-level roadmap and phase breakdown.
- `PROJECT_CONTEXT_REPORT.md` – Context and architecture report for the current implementation.
- `QUICK_START_GUIDE.md` – Quick start instructions for running the app.
- `STEP_6_VISITOR_PREAPPROVAL_COMPLETE.md` – Technical summary of the visitor pre-approval feature.
- `VISITOR_PREAPPROVAL_QUICK_START.md` – Focused quick start guide for visitor pre-approval flows.
- `firestore.rules` – Firestore security rules for the project (ready to deploy).
- `firestore-rules-fixed.rules` – Alternate/experimental Firestore rules variant.
